# Output Standards Rules

## Severity Rating

- Every security finding MUST include a severity rating: CRITICAL / HIGH / MEDIUM / LOW / INFO.

## Recommendations Format

- Every recommendation MUST include: action, owner, priority, and deadline.

## Report Structure

- Reports follow the structure: Executive Summary → Findings → Recommendations → Appendix.

## Language

- Use Vietnamese for internal reports unless English is explicitly requested.
- Technical terms (MITRE ATT&CK IDs, CVE IDs, CVSS scores) stay in English — they are
  international standards.
