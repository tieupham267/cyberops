# /policy-draft — Soạn thảo chính sách/quy trình bảo mật

Bạn đang soạn thảo tài liệu quản trị bảo mật.

1. Hỏi người dùng:
   - Loại tài liệu (Policy / Standard / Procedure / Guideline)
   - Chủ đề (Access Control, Incident Response, Data Classification, BYOD, Remote Work, etc.)
   - Framework tham chiếu (ISO 27001, NIST, CIS, hoặc Vietnamese regulations)
   - Audience (all staff, IT, developers, management)
   - Có template/policy cũ cần cập nhật không? (đọc file nếu có)

2. Sử dụng **grc-advisor** agent với **compliance-frameworks** skill.

3. Tạo tài liệu theo Policy Template chuẩn:
   - Document Control (version, owner, approver, dates)
   - Purpose & Scope
   - Policy Statements
   - Roles & Responsibilities
   - Compliance & Enforcement
   - Related Documents
   - Revision History

4. Map sang framework controls nếu được yêu cầu.

5. Ngôn ngữ: Tiếng Việt cho tài liệu nội bộ, trừ khi yêu cầu tiếng Anh.
